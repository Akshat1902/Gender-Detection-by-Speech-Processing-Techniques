Run gendetect.m

Select appropriate Option

1) Choose Database & Calculate Recognition Rate
 
Select collection of male/female wav files from the wav >> male/female folders

After selection of the database the system performs the gender detection on the wav files.

2) Recognize from audio file

Click on Start/Plot Signal

Select any wav file from your computer or from provided database, after the file is played click on Estimate Gender which gives the result.

3) Recognize by recording

Set the timer value to set the amount of time to record the speech of the speaker. After recording you can play to check the recorded file and plot signal, estimate gender to get the Result.
